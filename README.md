# AWS S3 Static Website Deployment

This project demonstrates the deployment of a publicly accessible static website using Amazon S3 with secure access configuration and infrastructure best practices.

---

## Architecture Overview

- Provisioned and configured an Amazon S3 bucket
- Enabled static website hosting
- Implemented a custom bucket policy for controlled public read access
- Applied least-privilege IAM principles
- Deployed and validated a live website endpoint

---

## Tech Stack

- Amazon S3
- IAM
- JSON (Bucket Policy)
- GitHub
